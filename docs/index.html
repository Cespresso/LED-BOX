🟥 LED BOX インストーラ

ブラウザから ESP32-C3 へファームウェアを書き込みます。ツールのインストールは不要です。

このブラウザは未対応です。Chrome または Edge を使用してください。 HTTPS 接続でのみ利用できます。

手順

  1. LED BOX を USB ケーブルで PC に接続
  2. 上の「⚡ インストール」を押す
  3. 表示されたシリアルポートを選択
  4. 書き込み完了を待つ（自動で再起動します）

デバイス情報

  • MCUESP32-C3
  • BLE デバイス名LED BOX
  • BLE パスキー123456
  • LED8x8 マトリクス (MAX7219)
対応ブラウザ： Chrome / Edge / Opera（デスクトップ版）。 Safari・Firefox・スマートフォンは Web Serial 非対応のため利用できません。
初回書き込み時はフラッシュ全体が消去されます（保存済みの設定はリセットされます）。
