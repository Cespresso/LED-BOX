🟥 LED BOX インストーラ

ブラウザから ESP32-C3 へファームウェアを書き込みます。ツールのインストールは不要です。

このブラウザは未対応です。PC は Chrome / Edge、スマホは Android の Chrome を使用してください（iOS・Safari・Firefox は非対応）。 HTTPS 接続でのみ利用できます。

手順

  1. LED BOX を USB ケーブルで接続
    • PC：Chrome / Edge にそのまま接続
    • Android スマホ：USB-OTG ケーブルで接続し Chrome でこのページを開く
  2. 上の「⚡ インストール」を押す
  3. 表示されたシリアルポート / USB デバイスを選択
  4. 書き込み完了を待つ（自動で再起動します）

書き込み後の使い方

  1. 「インストール完了」後に表示される空のダイアログは閉じて OK です （仕様です。デバイスはすでに起動しています）
  2. LED が点灯し、初期モード（Pomodoro）のアイコンが表示されます
  3. スマホ / PC の Bluetooth から LED BOX ****（末尾 4 文字は個体ごとに異なる）に接続。 ペアリングのパスキーは 123456
  4. 本体の 2 つのボタンで操作：
    • 白を長押し：モード切替（Pet → Pomodoro → Tools → Notification → SmartHome → Monitor → …）
    • 赤 / 白を短押し：各モードでの操作
  5. 明るさ・モードは BLE 経由でも変更できます

デバイス情報

  • MCUESP32-C3
  • BLE デバイス名LED BOX ****
  • BLE パスキー123456
  • LED8x8 マトリクス (MAX7219)
対応環境： デスクトップの Chrome / Edge / Opera（Web Serial）に加え、 Android の Chrome（USB-OTG / WebUSB）でも書き込めます。 iOS（iPhone / iPad）と Safari・Firefox は非対応です。
Android では充電専用ではなくデータ通信対応の OTG ケーブルを使用してください。
初回書き込み時はフラッシュ全体が消去されます（保存済みの設定はリセットされます）。
